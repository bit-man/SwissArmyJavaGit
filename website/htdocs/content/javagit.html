Welcome to the JavaGit project.

A Java API to provide java programs native access to git functionality. Get the Code We wanted to use a java API interface for git so that we could program natively in java and access the powerful features that git provides. Our goal is to provide java programmers with a well designed java API that will allow java developers to get fast and easy access to git features.

Get the Code:

	svn checkout https://subversive.cims.nyu.edu/osp/javagit/trunk/

Requirements:

  • OS: Windows, OS X, Linux , or possible other untested Unix systems
  • Software: Git (available through cygwin on windows)
